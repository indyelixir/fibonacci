# Fibonacci

An Elixir interface to the Fibonacci series.

(Created as an illustration for a talk at [Indy Elixir](http://www.indyelixir.org/))

## Installation

Add `fibonacci` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:fibonacci, "~> 0.1.0"}]
end
```

And run

```shell
$ mix deps.get
```
